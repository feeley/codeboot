Console

Enter a command at the console and then:

Enter / Shift Ctrl Enter / F7 : execute it fully and display result (unless command is incomplete)
F6 : animate execution of command (single steps repeated automatically)
Shift Enter / F5 : execute the first step (continue with Shift Enter or Enter)
↑ / Ctrl P : move back in command history or cursor up
↓ / Ctrl N : move forward in command history or cursor down

Editor

When the cursor is in an editor:

Enter : insert line break, move to next line and smart indent
Shift Ctrl Enter / F7 : execute the current file
F6 : animate execution of current file (single steps repeated automatically)
Shift Enter / F5 : execute the first step of current file (continue with Shift Enter or Enter)
↑ / Ctrl P : move cursor up
↓ / Ctrl N : move cursor down

Console and Editor
Esc / F8 : stop the execution (reset execution when repeated 3 times)
Ctrl L : clear the console
← / Ctrl B : move cursor left
→ / Ctrl F : move cursor right
Alt ← / Ctrl A : move cursor to beginning of line
Alt → / Ctrl E : move cursor to end of line
Ctrl J : insert line break and move to next line
Ctrl O : insert line break

Importing a file from the desktop is done using drag-and-drop.

Saving a file to the desktop is done using the download icon next to its name in the Files menu.

Renaming a file is done by double clicking its editor tab.
